# Installation

Neural Studio requires Tensorflow&gt;=2.2.0 to work. Other dependencies are as following.

* OpenCV
* Numpy
* Pandas
* Matplotlib
* Pyrex
* Scikit-learn
* Psutil
* GPUtil

{% hint style="warning" %}
Check for dependency version conflicts before installing neural studio.
{% endhint %}

#### Install From PyPi

```bash
pip install neural-studio
```
